# golang-webhook

To run webhook server:-
1. RUN 'go build' from 'golang-webhook/src/github.com/bajratech/webhook' path.
2. RUN 'go install' from 'golang-webhook/src/github.com/bajratech/webhook' path.
3. Replace 'bin/start.sh' file with 'golang-webhook/src/github.com/bajratech/webhook/start.sh'
4. RUN './webhook >> log.txt &'
